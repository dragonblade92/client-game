User Stories
Als speler wil ik een lobby kunnen maken zodat ik met anderen online kan spelen. Success
Als speler wil ik een lobby kunnen joinen zodat ik met anderen online kan spelen. Success
Als speler wil ik in real-time kunnen zien wat mijn tegenstander doet zodat ik daarop kan anticiperen. Success
Als speler wil ik het spel in veelgebruikte browsers kunnen spelen zodat ik zelf kan kiezen waarmee ik het spel speel. Success
Als speler wil ik kunnen zien welke lobbies er zijn zodat ik deze kan joinen. Success
Als speler wil ik mijn naam kunnen kiezen zodat ik mijn identiteit kan uiten. Success
Als speler wil ik in real-time mijn eigen personage zien zodat ik zie wat ik doe. Success
Als speler wil ik een splashscreen zien als ik de game start zodat ik zie wat voor spel ik start. Success
